# عيادات المشفى

نظام حجوزات ومتابعة دور احترافي جاهز للنشر على GitHub Pages وFirebase.

## التعديلات النهائية

- النظام يبدأ فارغًا تمامًا بدون أسماء أطباء أو مرضى أو حجوزات تجريبية.
- إضافة وتعديل وحذف الأطباء.
- إضافة وتعديل وحذف المرضى.
- إضافة وتعديل وحذف الحجوزات.
- مسح كل البيانات من صفحة الإعدادات.
- دعم أكثر من طبيب يعمل في نفس الوقت.
- دعم أكثر من غرفة، وكل طبيب مرتبط بغرفته.
- لكل طبيب بادئة أكواد خاصة، ويتم إنشاء كود مستقل لكل مريض تابع لهذا الطبيب مثل: `DR1-0001`.
- لكل طبيب قائمة انتظار مستقلة ورقم دور مستقل.

## التشغيل

```bash
npm install
npm run dev
```

## بناء نسخة النشر

```bash
npm run build
```

سيتم إنشاء نسخة النشر داخل مجلد `dist`.
